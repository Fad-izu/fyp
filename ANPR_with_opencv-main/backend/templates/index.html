About Us

Smart IKM Parking is an innovative system created by students of Institut Kemahiran MARA Besut. It combines Artificial Intelligence (AI) and Internet of Things (IoT) technologies to manage parking efficiently. Using YOLOv8, EasyOCR, and cloud integration, the system ensures accurate license plate detection and secure gate automation.

Features

AI Recognition

Automatically detects and recognizes license plates using YOLOv8 + EasyOCR models.

Cloud Integration

Student and vehicle data are securely stored in the cloud for instant access and analysis.

IoT Automation

Integrated with Raspberry Pi and Arduino for real-time gate automation and monitoring.
